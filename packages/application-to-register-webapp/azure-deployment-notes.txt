Using docker compose to deploy to azure container instances

https://docs.microsoft.com/en-us/azure/container-instances/tutorial-docker-compose

docker context use default

# build images for service
docker-compose -f services.build.yml build

# push image to azure container registry
docker-compose -f services.build.yml push

cd packages/application-to-register-webapp

docker context use bnglocaldev

docker compose up
